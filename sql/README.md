This directory contains the SQL projects (Unit 1, Lesson 2) from the Thinkful Data Science Bootcamp.

Projects 2, 3, and 4 use the [Bay Area Bikeshare dataset for September 2015 to August 2016](https://www.dropbox.com/s/4n7ocet119ul611/bikeshare_v2.db.zip?dl=0).

Project 5 uses the Denver summary listings and calendar csv's from the [Inside Airbnb dataset](http://insideairbnb.com/get-the-data.html).
